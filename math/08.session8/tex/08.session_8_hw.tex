Differentiate the following functions:

\begin{enumerate}
\item $y = \frac{2^x}{x + 1}$
\item $y = 7^{\sqrt{x^2 + 1}}$
\item $y = \log_{10}(2^x)$
\item $y = 3^{x^3 + 2x} \cdot 5^{x^2 - 1}$
\item $y = (\sin(x))^{2^x}$ (Hint: Use logarithmic differentiation)
\end{enumerate}
